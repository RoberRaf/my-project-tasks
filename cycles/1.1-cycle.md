# Technical Constraints (Cycle 1.1) [OUTCOME]

- **Performance:** API p95 latency <500ms under projected Beta load. Curriculum responses paginated (≤100 items/page) to protect 2GB RAM clients.
- **Security:** Session tokens stored server-side with secure signing; client stores in OS keychain/keystore (iOS Keychain / Android Keystore). HTTPS enforced. Rate limiting per user_id and per phone number. OTP codes hashed in storage. Anonymized deletion record complies with data retention per SOW §2.1.
- **Region:** AWS me-south-1 (Bahrain) for lowest latency to Iraqi users.
- **Offline-First Contract:** Sync endpoint designed for bidirectional push/pull with last-write-wins conflict resolution (SOW §2 offline-first architecture).
